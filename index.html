📦 Installation

  1. Download the .dmg file from the button above
  2. Open the DMG and drag Helm ADE to your Applications folder
  3. First launch: Right-click the app → Open (macOS may show a Gatekeeper prompt the first time)
  4. After the first launch, open normally from Applications or Spotlight

🔄 Auto-Updates

✨

Helm ADE checks for updates automatically using Sparkle. You can also check manually via Helm ADE → Check for Updates... in the menu bar.
